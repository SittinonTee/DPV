# การเชื่อมต่อเซนเซอร์ UV (VEML6075) กับ Waveshare ESP32-S3-Touch-LCD-4.3

บอร์ดที่คุณใช้ (รุ่นไม่มี Touch) มีพอร์ตขยายสำหรับเซนเซอร์ (Sensor Port) อยู่ด้านข้างบอร์ด ซึ่งสะดวกต่อการเชื่อมต่อมากครับ โดยเราจะใช้พอร์ตนี้แบบ I2C

## 1. ผังการต่อสาย (Wiring)

เซนเซอร์ DFRobot VEML6075 มี 4 พิน ให้ต่อเข้ากับ **Sensor Port** (คอนเนคเตอร์ 4 พินสีขาวบนบอร์ด) ดังนี้:

| พินเซนเซอร์ (VEML6075) | พินบนบอร์ด (Sensor Port) | รายละเอียด |
| :--- | :--- | :--- |
| **VCC** | **Pin 1 (3.3V)** | ไฟเลี้ยง 3.3V |
| **GND** | **Pin 2 (GND)** | กราวด์ |
| **SDA** | **Pin 3 (GPIO 6)** | ข้อมูล I2C |
| **SCL** | **Pin 4 (GPIO 5)** | สัญญาณนาฬิกา I2C |

> [!TIP]
> ลำดับพินบนบอร์ดมักจะมีตัวเลข 1 กำกับอยู่ข้างๆ หรือดูจากด้านหน้าจอ พิน 1 จะอยู่ด้านซ้ายสุดของช่องเสียบ

## 2. การเตรียมซอฟต์แวร์

คุณต้องติดตั้ง Library สำหรับเซนเซอร์ตัวนี้ใน Arduino IDE ก่อนจึงจะคอมพายล์ผ่าน:
1. ไปที่ **Tools** -> **Manage Libraries...**
2. ค้นหาคำว่า **"DFRobot_VEML6075"**
3. กด **Install**

## 3. รายละเอียดใน Code

- ไฟล์ `sensors.h` ถูกตั้งค่า `PIN_I2C_SDA` เป็น **6** และ `PIN_I2C_SCL` เป็น **5** เรียบร้อยแล้ว
- ไฟล์ `sensors.cpp` มีฟังก์ชันการอ่าน UVA, UVB และ UV Index (UVI) พร้อมส่ง Log ออกทาง Serial เพื่อตรวจสอบการทำงาน
- หากต้องการใช้ค่าจริงบนหน้าจอ ให้เปลี่ยนใน `ui.ino` จาก `tester_update()` เป็น `sensors_update()`

## 4. ข้อควรระวัง
เนื่องจากบอร์ดรุ่นนี้ใช้ GPIO 8 และ 9 สำหรับระบบภายใน (I/O Expander) การใช้พอร์ต Sensor (GPIO 5, 6) จึงปลอดภัยที่สุดและไม่รบกวนระบบเดิมของจอครับ
