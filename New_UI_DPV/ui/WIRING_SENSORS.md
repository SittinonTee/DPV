# คู่มือการต่อสายเซนเซอร์ BNO055 และ BMP280 (ละเอียด)

สำหรับบอร์ด Waveshare ESP32-S3-Touch-LCD-4.3 (รุ่นไม่มี Touch) เราจะใช้พอร์ตขยาย **Sensor Port** (คอนเนคเตอร์ 4 พินสีขาว)

## 1. ผังการเชื่อมต่อแบบพ่วง (I2C Daisy Chain)

การต่อแบบ I2C คือการนำพินที่มีชื่อเหมือนกันมาต่อรวมกันเป็นเส้นเดียว แล้วค่อยเสียบเข้าบอร์ดครับ

### แผนผังการต่อ (ASCII Diagram)
```text
[ Waveshare Board ]           [ BNO055 Sensor ]           [ BMP280 Sensor ]
  (Sensor Port)                 (I2C Module)                (I2C Module)
  
  Pin 1 (3.3V)  <----------->     VIN / VCC      <----------->     VCC
  Pin 2 (GND)   <----------->     GND            <----------->     GND
  Pin 3 (SDA)   <----------->     SDA            <----------->     SDA
  Pin 4 (SCL)   <----------->     SCL            <----------->     SCL
```

---

## 2. ขั้นตอนการต่อสายแบบทีละ step

1. **เตรียมสายไฟ**: คุณต้องมีสายที่เสียบเข้ากับพอร์ตสีขาว (มักจะเป็นสาย 4 เส้น: แดง, ดำ, เหลือง, น้ำเงิน)
2. **รวมสายไฟ (Daisy Chain)**:
   - นำสายพิน **SDA** จาก BNO055 และ BMP280 มาบัดกรีหรือต่อรวมกัน แล้วเชื่อมเข้ากับ **Pin 3 (GPIO 6)** ของบอร์ด
   - นำสายพิน **SCL** จาก BNO055 และ BMP280 มาบัดกรีหรือต่อรวมกัน แล้วเชื่อมเข้ากับ **Pin 4 (GPIO 5)** ของบอร์ด
   - ทำเช่นเดียวกันกับสาย **VCC (3.3V)** และ **GND**
3. **ตรวจสอบแรงดันไฟ**: ตรวจสอบให้แน่ใจว่าใช้ไฟ **3.3V** เท่านั้น (ห้ามต่อเข้า 5V เพราะอาจทำให้อุปกรณ์เสียหาย)

---

## 3. การระบุพินบนบอร์ด (Sensor Port)

หากคุณมองบอร์ดจากด้านหน้าจอ (พอร์ตอยู่ด้านบนหรือขอบข้าง):
- **พิน 1**: มักจะมีตัวเลข `1` กำกับ หรือเป็นพินที่อยู่ชิดขอบด้านใดด้านหนึ่ง (ตามสถาปัตยกรรม Waveshare คือพินซ้ายสุด)
- ลำดับคือ: `[ 1:3.3V | 2:GND | 3:SDA(6) | 4:SCL(5) ]`

---

## 4. หลังจากการต่อสาย
1. เปิด Serial Monitor ใน Arduino IDE (ความเร็ว 115200)
2. หากต่อถูกต้อง คุณจะเห็นข้อความ:
   - `BNO055 Connected.`
   - `BMP280 Connected.`
3. หากขึ้นว่า `Failed`, ให้ลองสลับสาย SDA และ SCL หรือตรวจสอบว่าพิน SDO ของ BMP280 ต่อลง GND หรือไม่ (เพื่อระบุ Address 0x76)
